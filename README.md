# Голосовой ассистент на Python
Сейчас Дилл понимает только голос и отвечает голосом. Умеет говорить время, погоду, искать в интернете информацию и включать видео.
В ближайшем обновлении добавлю текстовый чат. Это значит:

Можно будет печатать команды в окне и получать текстовые или голосовые ответы

Ассистент станет удобнее для работы за компом — не нужно говорить вслух

Появятся новые фичи вроде быстрых заметок, перевода текста и суммаризации статей

Следите за обновлениями — будет интересно.
